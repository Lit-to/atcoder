# 言語差用語変換シート


|   No |      項目      | Python                    | C++                | Java       |
| ---: | :------------: | :------------------------ | :----------------- | :--------- |
|    1 |   固定長配列   | (該当なし)                | std::Array         | ?          |
|    2 |   可変長配列   | **List**                  | std::vector        | ArrayList  |
|    3 |  双方向リスト  | Dequeで代用               | **std::List**      | LinkedList |
|    4 |     キュー     | Dequeで代用               | std::Queue         | Queue      |
|    5 |    スタック    | Dequeで代用 or Listで代用 | std::Stack         | stack      |
|    6 | 順序なし集合型 | **Set**                   | std::unordered_set | HashSet    |
|    7 | 順序つき集合型 | SortedSet                 | **std::set**       | ?          |
|    8 | 順序なし辞書型 | Dict                      | std::unordered_map | HashMap    |
|    9 | 順序つき辞書型 | SortedDict                | **std::map**       | ?          |

-   言語間で用語の意味が異なるところには強調を付けている

> ※1 PythonでMapと言った場合はmap関数のことを指すため注意
