## シェル使い方メモ
最終更新:2026-04-11 06:55:11

-   ``hide``
    -   →精進に不要なdev/modules/sandBoxフォルダとREADME.mdを隠す
-   ``show``
    -   隠したdev/modules/sandBoxフォルダとREADME.mdを表示する
-   ``move``
    -   ``dev``配下にあるA?C<数字三桁><A~Z>のファイルを``submission``フォルダに移動
-   ``practice <問題名>.言語拡張子``
    -   modules配下からtemplate.言語拡張子のファイルをコピーしてリネームして配置
-   ``start <コンテスト名>``
    -   一気にA~Gまでのファイルを``dev``フォルダに配置
-   ``attest``
    -   test配下にあるすべてのファイルを実行
